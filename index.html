Taskmaster UK Dashboard

In this project I created a dashboard showing graphs of various stats from the TV series Taskmaster. I used the R package rvest to scrape the data from the website https://taskmaster.info created by Karl Craven and created a dashboard using the package flexdashboard. You will find graphs showing data from all series combined as well as a page for each individual series. Your time starts now!
Github repo can be found here.
